# THE BEST CITY for me! 🧭✨

An immersive, high-end bilingual quiz that discovers which perfect city in the world matches your personality, preferences, and lifestyle vibes.

---

## 🇧🇷 Português

### Sobre o Projeto
O **THE BEST CITY for me!** é um quiz interativo desenvolvido em **React**, **Vite** e **Tailwind CSS**. Através de 15 perguntas dinâmicas cobrindo clima, custo de vida, ritmo urbano, hobbies e cultura, o motor calcula a compatibilidade com cidades globais e apresenta um destino principal em formato Bento Grid, acompanhado de alternativas secundárias.

### Características Principais
- **Design Imersivo:** Layout ambientado no espaço, com poeira estelar responsiva em CSS e motion animations fluidas via `motion`.
- **Bilinguismo Completo:** Alternância de idioma com um clique (Português & Inglês).
- **Sem Dependências Pesadas:** Totalmente customizado para gerar arquivos prontos para o GitHub Pages (`index.js` e `index.css` diretamente na pasta `/dist`).
- **Desenvolvido por:** Nicolas Cailan.

## 🇬🇧 English

### About the Project
**THE BEST CITY for me!** is an interactive, high-end bilingual quiz built using **React**, **Vite**, and **Tailwind CSS**. Through 15 dynamic questions spanning climate, cost of living, urban pacing, hobbies, and culture, the engine calculates your personal compatibility index with global hubs, rendering a custom Bento Grid highlight of your top city and secondary options.

### Key Features
- **Immersive Design:** Cosmic slate theme with lightweight interactive stellar elements and smooth, physics-guided micro-transitions powered by `motion`.
- **Fully Bilingual:** Seamless UI switching with a floating header switcher (Portuguese & English).
- **GitHub Pages Ready:** Built with customized build configurations delivering compiled assets (`index.js` and `index.css` directly into `/dist`) for direct hosting.
- **Developed by:** Nicolas Cailan.

## 🛠️ Instalação e Desenvolvimento / Installation & Development

### Pré-requisitos / Prerequisites
- Node.js (v18+)
- npm

### Passos / Steps

1. Instale as dependências / Install dependencies:
   ```bash
   npm install
   ```

2. Rode em modo de desenvolvimento / Run the development server:
   ```bash
   npm run dev
   ```

3. Compile o build de produção / Compile production build:
   ```bash
   npm run build
   ```
   *Após a compilação, todos os arquivos necessários para o GitHub Pages serão criados de forma simples e limpa dentro do diretório `/dist`.*
